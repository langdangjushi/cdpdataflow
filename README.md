# cdpdataflow
dataflow engine based on Apache Flink

## 编译打包
* 推荐使用maven 3.2.5版本
* mvn clean install

## Flink Job dispatcher
emit SQL to flink cluster management system(standalone / YARN).

* [Postgres](./dataflow-connectors/postgres.md)
* [HBase](./dataflow-connectors/hbase.md)
